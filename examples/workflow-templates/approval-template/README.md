# 审批工作流程模板

这是一个多级审批工作流程模板，包含初审和终审两个审批环节。

## 使用方法

1. 复制此目录到您的项目位置
2. 修改 `datamodel.cm` 中的项目信息
3. 根据需要调整审批流程

## 项目结构

```
approval-template/
├── datamodel.cm              # 数据模型定义
├── workflows/                # 工作流程目录
│   └── ApprovalWorkflow.workflow.cm  # 审批工作流程
└── README.md                 # 项目说明
```

## 流程说明

```
提交申请 → 初审 → 初审决策 → 终审 → 终审决策 → 审批通过
                    ↓                    ↓
                审批拒绝 ←←←←←←←←←←←←←←←←←
```

### 节点说明

| 节点 | 类型 | 说明 |
|------|------|------|
| 提交申请 | 开始节点 | 申请人提交申请 |
| 初审 | 过程节点 | 第一级审批人审核 |
| 初审决策 | 分支节点 | 初审是否通过 |
| 终审 | 过程节点 | 最终审批人审核 |
| 终审决策 | 分支节点 | 终审是否通过 |
| 审批通过 | 结束节点 | 申请已批准 |
| 审批拒绝 | 结束节点 | 申请被拒绝 |

### 泳道说明

| 泳道 | 包含节点 |
|------|----------|
| 申请人 | 提交申请 |
| 初审人 | 初审、初审决策 |
| 终审人 | 终审、终审决策 |

## 自定义

### 添加更多审批级别

1. 在 `nodes` 中添加新的过程节点和分支节点
2. 在 `edges` 中添加相应的连接
3. 在 `swimlanes` 中添加新的泳道

### 修改审批条件

修改分支节点的 `branches` 属性来调整审批条件。
